## Skills instaladas para el equipo

Las siguientes capacidades de Claude Code están instaladas en `.claude/skills/` del proyecto y `.claude/commands/`:

- **research-grants** — escritura de propuestas tipo NSF/NIH/DOE
- **scientific-writing** — IMRyD, prosa académica
- **peer-review** — revisión sistemática
- **literature-review** — búsqueda en PubMed/arXiv/Semantic Scholar
- **citation-management** — BibTeX, Google Scholar
- **scholar-evaluation** — framework ScholarEval cuantitativo
- **scientific-critical-thinking** — GRADE, Cochrane ROB, sesgos
- **academic-paper** — pipeline 12 agentes para escritura
- **academic-paper-reviewer** — 7 agentes multi-perspectiva
- **academic-pipeline** — orquestador research→write→review
- **deep-research** — 13 agentes para investigación profunda
- `/review-grant` — slash command para panel pre-submission de 6 agentes

→ Ejemplo de uso: en la próxima sesión de iteración, puedes correr `/review-grant major-funder propuesta_v2_consolidada.md` para hacer una pasada de revisión panel sobre la versión que estemos trabajando.
